# Check-in Nova Aliança

Sistema de check-in desenvolvido em Flask para controle de presença de:
- Visitantes (com dia da semana)
- Voluntários (com setor)
- Obreiros(as)

## Tecnologias
- Python 3
- Flask
- SQLite
- HTML + CSS (degradê verde → ciano)

## Como rodar o projeto

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
